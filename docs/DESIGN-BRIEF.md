# Vocast 랜딩 페이지 디자인 브리프

> 디자인 에이전트/디자이너 핸드오프 문서. 이 파일 + 라이브 페이지 + 리포지토리를
> 함께 전달하면 바로 작업 착수 가능.

## 제품 한 줄

크리에이터를 위한 **로컬(온디바이스) Mac 음성 스튜디오**. 노이즈 제거 + 보이스
클로닝. **원타임 구매 · 1년 무료 업데이트 · AI 에이전트(MCP) 내장.**

## 목표

Raycast의 "프로 개발자 도구" 무드를 차용하되, "음성/크리에이터"에 맞게 따뜻하게
변주한 세일즈 랜딩. 최종 산출물은 Figma 파일(데스크톱 1440 + 모바일 390) 풀 디자인.

## 훔쳐올 Raycast DNA (반드시 지킬 것)

1. **네이티브 앱 같은 다크 캔버스** — 순수 블랙이 아니라 아주 살짝 블루가 도는
   near-black(#07080a). "웹사이트"가 아니라 "macOS 앱 안"에 있는 느낌.
2. **여백이 곧 럭셔리** — 섹션마다 120px+ 세로 여백. 요소를 욱여넣지 않는다.
   한 화면에 한 메시지.
3. **2톤 헤드라인** — 첫 줄은 흰색 단언, 둘째 줄은 회색 부연.
   (예: "구독이 아닙니다."(흰) / "한 번 사면 당신 것."(회))
4. **액센트는 '찍는' 것이지 '칠하는' 것이 아니다** — 컬러는 CTA·키커·글로우에만.
   90%는 무채색, 10%만 액센트.
5. **글래스 + 레이어드 섀도우** — 플로팅 필 내비, 카드에 1px 하이라인
   보더(rgba white .08~.14) + macOS식 다층 그림자(inset 하이라이트 포함).
   "다크 책상 위에 놓인 유리" 질감.
6. **저강도 라디얼 글로우(Nebula)** — 히어로·CTA 뒤에 낮은 불투명도의 발광.
   색광이 흑요석 뒤에서 새어나오는 느낌.
7. **에어리한 타이포** — Inter 계열, body에 약간의 양수 letter-spacing(+0.1~0.4px)으로
   다크에서도 답답하지 않게. 헤드라인은 tight(-0.03em).

## 우리만의 변주 (Raycast와 다르게)

- **웜 코럴 액센트**: Raycast의 시그니처는 레드(#FF6363)+쿨블루 글로우지만, 우리는
  **핑크→코럴→오렌지 그라디언트**(#ff5c8a → #ff6b5b → #ff9159)로 간다.
  이유: "목소리·따뜻함·크리에이터"에 쿨톤보다 웜톤이 맞음. 글로우도 핑크/오렌지 계열.
  (더 나은 방향 제안 환영 — 단, 웜 계열 유지)
- **한글 타이포**: Inter 대신 **Pretendard**(한글) + Inter(숫자/영문) 혼용.
  헤드라인 800 웨이트.
- "장난기"는 Raycast보다 절제. 우리는 프로 크리에이터 도구.

## 페이지 구조 (섹션별)

1. **플로팅 글래스 필 내비** — 로고 + 앵커링크 + GitHub(고스트) + 구매(그라디언트)
2. **히어로** — 파일럿 필(원타임·로컬·MCP) → 2톤 대형 헤드라인(액센트 라인은
   그라디언트 텍스트) → 부제 → CTA 2개 → **인터랙티브 A/B 오디오 비교 카드**(원본↔
   제거 후 토글, 아래 품질 수치 칩). 뒤에 코럴 글로우 오라.
3. **문제 제기** — 3카드("말끝 사라짐", "재녹음 지옥", "구독·업로드 불안")
4. **기능 그리드** — 6개 카드, 각 카드에 앱 실제 화면 미니 목업. 호버 시 상단
   그라디언트 라인 점등. + 하단에 **가라오케 가사 뷰 라이브 데모**(단어가 재생
   위치 따라 색칠).
5. **품질 신뢰 블록** — 4개 지표 스탯 카드(그라디언트 숫자) + "방법론 공개" 링크.
   ("감으로 안 만들었다"는 차별화 메시지)
6. **로컬-퍼스트** — 3포인트(업로드 없음 / 오프라인 / 내 파일은 내 것)
7. **MCP 섹션 (핵심 차별화)** — "Claude가 이 앱을 직접 조종합니다." Claude↔앱 채팅
   목업, 툴 호출 로그는 모노스페이스로. **시장에서 거의 유일한 포지셔닝이니
   시각적으로 가장 임팩트 있게.**
8. **가격** — 코럴 글로우 보더의 단일 카드. ₩59,000 원타임, 체크리스트 5개, "구독
   아님" 강조. Raycast 가격 페이지 톤 참고.
9. **FAQ** — 아코디언(+/– 액센트)
10. **최종 CTA + 푸터**

## 컴포넌트 상태 (전부 그릴 것)

- **버튼**: primary(그라디언트 + 글로우 섀도우) / ghost, hover(y-1px lift, 글로우
  강화), sm 사이즈
- **카드**: default / hover(보더 밝아짐 + 상단 그라디언트 라인 + y-2px)
- **오디오 플레이어**: 다크 테마에 맞춘 스타일(네이티브 컨트롤 반전 or 커스텀)
- **A/B 토글 버튼**: off(무채) / on(코럴 글로우 인셋)
- **가라오케 단어**: 미래(faint) / 지난(흰색) / 현재(코럴 + 글로우 텍스트섀도우)
- **반응형**: 1440 데스크톱 → 390 모바일. 필 내비는 모바일에서 햄버거.

## 하지 말 것

- 순백 배경 섹션(라이트 모드로 튀는 블록) 금지 — 통일된 다크.
- 액센트 컬러 남발 금지(면적 10% 룰).
- 스톡 일러스트/이모지 남발 금지. 제품 실제 UI 목업이 주인공.
- 그라디언트를 큰 면적 배경으로 쓰지 말 것(글로우/텍스트/버튼에만).

## 산출물

- Figma: 데스크톱 + 모바일 전 섹션, 컴포넌트는 오토레이아웃 + 배리언트,
  컬러/타이포 스타일 토큰화.
- 인터랙션 노트(호버 · A/B 토글 · 가라오케 동작)는 코멘트로.

## 참고: 제품 팩트 시트 (카피/디테일 근거)

디자인 목업 안 텍스트나 수치를 채울 때 쓸 실제 스펙:

- **노이즈 제거**: 표준(필터 하이브리드) / 재합성(생성형) 2모드. 발화 손실 0.0%
  (말끝 보존), 무음 소음 억제, 목소리 유사도 0.85+ 게이트.
- **보이스 클로닝**: 가이드 문장 10개로 프로필 생성 → 소스 추가로 강화(버전 관리·롤백).
  2만 자 원고, 문단 단위 재생성, 직접 연기 반영.
- **품질 시스템**: PNS(운율 자연스러움) 북극성, 말끝 보존 게이트, 78개 자동 테스트,
  결과물마다 품질 리포트. 방법론 전체 공개(QUALITY.md).
- **로컬**: 서버 없음, 전 처리 온디바이스, 오프라인 동작(모델 최초 다운로드만 온라인).
- **MCP**: 앱 전 기능을 로컬 MCP로 노출 → Claude·Codex가 대화로 앱 조작.
- **가격**: ₩59,000 원타임 · 1년 무료 업데이트 · 이후 갱신 선택(₩29,000/년) · Mac
  3대까지 · 14일 환불.
- **요구사항**: macOS Apple Silicon(M1+), 16GB 권장.
